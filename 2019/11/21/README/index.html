How to create a blog post

  1. Create a markdown file in the source/_posts dir.
  2. Fill your content.
  3. Create a test branch and create PR on this repo.
  4. Wait the pr workflow success.
  5. After the administrator of toc blog approved and merged, your blog will be displayed on our blog system.

How to change a theme

Currently, the theme is cloned from github. To use another theme, you need:

  1. Change the theme repo url in Prepare theme of .github/workflows/main.yml.
  2. Change the theme in _config.yml.
